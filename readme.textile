h1. Installation:

<pre><code>git clone git://github.com/squiter85/dotfiles.git ~/dotfiles
cd ~/dotfiles
sh install.sh
</code></pre>

Initialize and update all submodules:
<pre><code>git submodule update --init
</code></pre>

Install Vim plugins with Vundle:
<pre><code>vim +PluginInstall +qall
</code></pre>

Set zsh as default shell
<pre><code>chsh -s /bin/zsh
</code></pre>

To add custom configurations just add it into @~/.zsh_custons@

To use Taglist plugin, just install ctgas with: @brew install ctags@
